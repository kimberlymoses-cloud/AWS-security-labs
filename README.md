# AWS-security-labs
Hands-on AWS security projects and labs covering IAM, S3 encryption, VPC security, KMS and more. Built for AWS Certified Solution Architect and Comptia Security+ cloud security skills. These labs include security focused AWS architecture diagrams, step by step lab guides (CLI + console), code snippets for automation, and threat modeling examples
